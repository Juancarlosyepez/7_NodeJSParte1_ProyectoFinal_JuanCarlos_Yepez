Examen Final de NodeJS Parte 1

NodeJS es una plataforma de software para crear aplicaciones entre cliente-servidor y servidor-servidor. Adicionalmente, es una plataforma que se instala en el Sistema Operativo y ejecuta su funcionalidad desde la máquina. La finalidad de la aplicación es simular el dibujo y edición de imágenes en tiempo real con más de un usuario haciendo uso deExpressJS; MongoDB y MongooseJS para la gestión de usuarios; Redis y PassportJS para el proceso de login; y finalmente implementar el concepto de Socket.IO para la edición en tiempo real de las imágenes.
